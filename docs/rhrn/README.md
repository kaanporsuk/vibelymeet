# RHRN Sprint 0 Docs Package

This folder contains the Sprint 0 architecture handoff for Vibely's Right Here Right Now feature.

Read in this order:

1. `rhrn-architecture-report.md` - bounded-context architecture, Core touchpoints, product constraints, risks, and open questions.
2. `rhrn-data-model.md` - proposed `rhrn_*` schema, RLS posture, security rules, and migration risk notes.
3. `rhrn-edge-functions-and-rpcs.md` - proposed API surface, auth posture, sanitized payload rules, notifications, payments, and Google Places boundary.
4. `rhrn-ui-map.md` - web/native screen map, states, settings, card/profile behavior, Vibe/Vibe Note flow, Teleport, and admin UI.
5. `rhrn-rollout-and-qa-plan.md` - feature flags, rollout, rollback, leak checks, QA matrix, rebuild requirements, and implementation sprint plan.

Sprint 0 is documentation only. Do not treat these files as implemented runtime behavior.
